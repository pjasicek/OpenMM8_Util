Installation
============

unpack Might&Magic Mapviewer  and "Ogre3D files" in some folder, and launch
on first launch it ask game folder (Might&Magic VI,Might&Magic VII or Might&Magic VIII)
and video setting
for change game folder delete mmview.cfg
for change video setting delete ogre.cfg

Using
============
F1: show help
ESC: quit
WSAD: Move
SHIFT: +speed
L: Load Map
PrtSc: Screenshot
R: Render mode
F: Toggle frame rate stats on/off
T: Cycle texture filtering(Bilinear, Trilinear, Anisotropic(8))
P: Toggle portals(BLV map only)
MOUSE2: select face/bmodel
H: show/hide selected BModel(odm map only)
Alt-H: show all hided BModels(odm map only)
Y: show/hide ents and spawns
